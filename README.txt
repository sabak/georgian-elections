src/ has the JuPyter notebooks. analysis.ipynb is just working script. presentation.ipynb is better structured and has clearer plots.
env/ is a python virtual environment with requirements.txt containing all the requirements.
plots/ contains all the outputs that are generated by presentation.ipynb file. This directory has one image which is taken from https://doi.org/10.1073/pnas.1210722109
data/ has all the election data that are used in the scripts. It even has more data that are not currently being analyzed.
